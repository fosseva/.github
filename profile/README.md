# Fosseva

**Open software. Reliable infrastructure.**

Fosseva builds open-source applications and helps teams run them with confidence through sponsorships, paid support, cloud setup, and managed hosting.

## What We Do

- Build practical open-source applications
- Maintain public projects with clear documentation
- Provide paid support for teams using our software
- Offer cloud setup, deployment, hosting, and maintenance
- Support sustainable open-source development through sponsorships

## Our Mission

We believe open-source software should be easy to inspect, easy to use, and reliable enough for real-world teams. Fosseva exists to build useful open tools and provide the professional support needed to run them in production.
